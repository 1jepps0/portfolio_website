Portfolio Website

A website built with django that utilizes an Obsidian plugin to send writeup information.

Writeups

On this page, writeups are automatically organized by competition and category. The competition and category can be further filtered using the drop-down menus at the top. The writeups are stored in a Django model which are created via a POST request from a custom Obsidian plugin. Obsidian is a Markdown note-taking tool that includes the ability to extend its functionality using a typescript plugin.

The markdown format of the application allows me to quickly create writeups that use code snippets and images. My plugin takes the content of the markdown and the images and sends a post request to my website.

Before sending the request, the plugin shows a prompt to collect the relevant information about the challenge. This allows me to sort and organize the writeups.

To convert the markdown into HTML to display to the page, I'm using a library called md-block. Additionally, I'm using prism paired with md-block to add syntax highlighting to my code-snippets.
